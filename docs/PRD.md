# PRD-001 记账 app V2

## 功能需求

### FR-01 H5 App 载体
- HTML5 + CSS3 + JavaScript 单页应用（SPA）
- 响应式布局，适配手机浏览器（优先）和桌面浏览器
- 通过 GitHub Pages 托管

### FR-02 数据存储（localStorage）
- 纯 localStorage 持久化，不依赖任何外部服务
- 用户数据存在浏览器本地，支持多用户（et_users / et_lastUser）
- 单机使用，不需要网络

### FR-03 账单管理（收入+支出）
- 支持手动新增账单记录
- 字段：金额、类型（收入/支出）、分类、备注、时间
- 支持查看账单列表（按时间倒序）
- 支持编辑和删除单条账单
- 支持日期选择（默认当天，可补录历史）

### FR-04 账单分类
- 预设支出分类 18 个：餐饮🍜、交通🚗、购物🛒、娱乐🎮、居住🏠、医疗🏥、教育📚、通讯📱、护肤💄、旅游✈️、运动健身💪、宠物🐶、社交🎁、烟酒🚬、投资📈、汽车🚙、母婴👶、其他📌
- 预设收入分类 6 个：工资💼、兼职💻、投资理财📈、被动收入🏠、红包🧧、其他📌
- 用户可新增/编辑/删除自定义分类（预设分类不可删除）

### FR-05 日报
- 统计范围：自然日（0:00–23:59），时区 GMT+8
- 数据：当日收入合计、支出合计、结余
- 图表：支出分类饼图（CSS conic-gradient）

### FR-06 月报
- 统计范围：自然月（1日–月末）
- 数据：当月收入合计、支出合计、结余、预算消耗进度
- 图表：支出分类饼图
- 交互：月份切换（上一月/下一月）
- 收支记录列表（按时间倒序）

### FR-07 月度预算提醒
- 用户设置月度预算上限（金额）
- 触发时机：当月累计支出达到预算的 80% 和 100% 时各弹窗提醒一次
- 每月重新计算（每月1日重置）
- 超出 100% 后不限制继续记录，仅提醒

## 用户故事
- 作为用户，我希望在手机上快速记录每一笔收支，随时随地记账
- 作为用户，我希望按日/按月查看收支汇总和分类分布，了解消费结构
- 作为用户，我希望设置月度预算并在接近/达到上限时收到提醒
